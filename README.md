# Call Analytics Dashboard

## Project Description

The **Call Analytics Dashboard** is a professional web application designed to visualize and analyze **Call Data Records (CDRs)** generated by telecommunication platforms. Every call generates a record containing key information such as caller and receiver details, call duration, status, timestamps, and cost.  

This dashboard helps businesses and call centers gain actionable insights from their call data by allowing them to:  
- **Track call activity** across different regions and time periods  
- **Analyze customer engagement** through successful and unsuccessful call trends  
- **Monitor call success rates** to identify operational issues  
- **Evaluate call costs** for budgeting and cost optimization  
- **Identify calling patterns** for better resource allocation and decision-making  

Built with modern frontend technologies and interactive data visualizations, this dashboard provides a **SaaS-style professional interface** that is both responsive and intuitive, enabling users to make data-driven decisions with ease.  

---

## Technology Stack

The project uses modern web technologies for a fast, responsive, and interactive experience.

### Frontend
- **React.js** – UI and component-based architecture  
- **Tailwind CSS** – Styling and responsive design  
- **JavaScript (ES6+)** – Logic and data handling  

### Charts & Visualization
- **Chart.js / Recharts** – Used for `DurationChart`, `CostChart`, and `ActivityTimeline`  

### Other Libraries / Tools
- **React Icons** – Icons for KPI cards and dashboard elements  
- **Fetch API** – For fetching call data from the CDR API  

### Deployment Options
- **Vercel** – Recommended for hosting and deployment
